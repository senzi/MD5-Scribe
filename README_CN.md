# MD5-Scribe

MD5-Scribe 是一个本地优先的 MD5 手算学习与校验工具。所有计算、任务进度和错误日志都在浏览器内完成，不需要 Python、数据库或后端服务。

## 功能

- 在浏览器中实现完整 MD5 算法，并记录每一轮的原子计算值。
- 将输入限制为 55 个 UTF-8 bytes，保证练习固定为单个消息块、64 轮。
- 提供每轮控制台、可打印工单和五个中间结果校验。
- 使用 `localStorage` 自动保存多个独立任务，支持随时创建、切换和删除挑战。
- 完成 64 轮后校验 A/B/C/D 的小端字节序和最终摘要。
- 在本地生成错误分析、时间线和审计日志，并下载 Markdown 报告。
- 应用运行期间不向服务器上传消息或进度。

## 开发运行

```bash
npm install
npm run dev
```

打开终端中 Vite 给出的本地地址，通常是 `http://localhost:5173`。

不要直接用普通静态服务器打开项目根目录的 `index.html`，它是 Vite 的开发入口。检查生产版本时请运行：

```bash
npm run build
npm start
```

## 测试与构建

```bash
npm test
npm run build
```

生产文件生成在 `dist/`。由于 `base` 使用相对路径，可以部署到任意静态文件托管服务。生产版本会注册 Service Worker；首次成功加载后，静态资源可离线打开，任务数据始终只保存在当前浏览器。

## 本地数据

任务工作区保存在浏览器的 `localStorage`，键名为 `md5-scribe:workspace:v3`。旧版单任务数据会自动迁移为第一个挑战。Markdown 报告通过浏览器下载，不会写入项目目录或上传到远端。

## 工单说明

每轮校验五个原子结果：

1. `T1 = A + f`
2. `T2 = T1 + M_j`
3. `T3 = T2 + K_i`
4. `ROT = RotateLeft(T3, s)`
5. `B_new = ROT + B`

打印按钮使用浏览器原生打印对话框，可以直接打印或另存为 PDF。第一轮工单附带十六进制与四位二进制对照表。
